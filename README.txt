INMO STUDIO PRO v3.5

Abrir index.html en Chrome.

CORRECCION PRINCIPAL
- Al seleccionar una imagen aparece obligatoriamente el panel Encuadre de imagen.
- Mover marco mueve/redimensiona el bloque.
- Mover foto mantiene el marco fijo y mueve la imagen interior.
- Rueda del mouse o control de zoom amplian/reducen la fotografia.
- Funciona con Lavinia, Pierina y Ruby/Lima Houses mediante el motor compartido.

Esta version usa editor-v35.css e image-crop-v35.js para evitar cache de versiones anteriores.
